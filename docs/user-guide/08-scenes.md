# 8. Scenes

[← Prev: Sampler Pads](./07-sampler-pads.md) · [Back to User Guide](./README.md) · [Next: YouTube Import →](./09-youtube-import.md)

---

A **Scene** is a saved snapshot of your entire board — what's loaded, what's
playing, and how everything is set. Recall a scene and DNDj rebuilds that exact
state in one click. Perfect for prepping set-pieces ahead of a session: "The
Tavern", "Ambush on the Road", "Final Boss".

Scenes live in the **bottom half of the left rail**, under the *SCENES* header.

## What a scene captures

When you save a scene, DNDj records:

- **Deck A, Deck B, and Deck C** — which track is loaded on each, plus its
  volume, filter setting, and loop points.
- **The crossfader** — position and curve.
- **The sampler pads** — which sound is on each of the eight pads.

## Saving a scene

1. Set up your board exactly how you want it — load decks, set the crossfader,
   assign pads.
2. In the *SCENES* panel, click **+ Save**.
3. Type a name and press **Enter** (or click ✓).

The new scene appears in the list with a **✦** snapshot icon.

## Recalling a scene

Each snapshot row has two recall buttons:

- **▶ (instant)** — rebuild the board immediately. Decks stop and reload to the
  saved state.
- **⟶ (fade)** — rebuild with a smooth ~2-second crossfade: the current decks
  fade out while the scene's decks fade in. Smoother for a live transition mid-game.

## Deleting a scene

Hover a scene row and click the **×** that appears.

## Tips

- Build a scene per major beat of your session in advance, then just hit recall
  when the players arrive there.
- Use the **fade** recall during play so the music morphs rather than jumps.
- Scenes store *references* to tracks (by their file path), so if a track's file
  is missing on the current machine, that deck simply won't load — the rest of the
  scene still recalls.

> **Two kinds of rows:** scenes you save in the Studio show the **✦** icon
> ("snapshots"). You may also see older **♪** "classic" scene rows if any exist in
> your database from earlier versions — these are display-only and can be deleted.
