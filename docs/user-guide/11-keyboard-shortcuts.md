# 11. Keyboard Shortcuts

[← Prev: Syncing Two Machines](./10-sync.md) · [Back to User Guide](./README.md) · [Next: Settings →](./12-settings.md)

---

DNDj is designed to be played by hand during a session. These shortcuts work
whenever you're not typing in a text box. The same list is available in-app under
**⚙ → Shortcuts**.

> The **"active deck"** is whichever of Deck A or Deck B you most recently
> started. Many shortcuts act on it, with `Shift` targeting the other deck.

## Decks

| Keys | Action |
|------|--------|
| `Space` | Play / Pause — active deck |
| `Shift` + `Space` | Play / Pause — other deck |
| `A` | Play / Pause — Deck A |
| `B` | Play / Pause — Deck B |
| `S` | Stop — active deck |
| `Shift` + `S` | **Stop ALL sounds** (decks + sampler) |

## Seek

| Keys | Action |
|------|--------|
| `←` | Seek −5 seconds |
| `→` | Seek +5 seconds |
| `Shift` + `←` | Seek −30 seconds |
| `Shift` + `→` | Seek +30 seconds |

## Loop

| Keys | Action |
|------|--------|
| `L` | Toggle loop on the active deck |

## Crossfader

| Keys | Action |
|------|--------|
| `[` | Nudge crossfader toward A |
| `]` | Nudge crossfader toward B |
| `\` | Centre the crossfader |
| Two-finger swipe | Adjust the crossfader (trackpad, hover over it) |

## Sampler (SFX) pads

| Keys | Action |
|------|--------|
| `1` – `8` | Trigger the matching pad |

## Waveform (mouse / trackpad)

| Input | Action |
|-------|--------|
| Click | Seek to that position |
| Mouse wheel | Zoom in / out |
| Pinch | Zoom in / out (trackpad) |
| Two-finger swipe | Pan when zoomed (trackpad) |
| Click & drag | Pan when zoomed |
