# DNDj User Guide

Welcome! This guide explains everything you can do with DNDj, written for the
person sitting at the table running a game — no coding knowledge needed.

If this is your first time, read the chapters in order. If you already know your
way around, jump straight to the feature you need.

## Core ideas in 60 seconds

- Your **sounds live in folders on disk** (inside the app's `sounds/` directory).
  Each subfolder is a **category** (e.g. `atmosphere`, `combat`, `sfx`).
- DNDj **scans** that folder and shows every audio file as a **track** in
  *My Library*.
- You **drag tracks onto decks** to play them with full DJ controls, onto
  **sampler pads** for instant SFX, or into **playlists** to organise them.
- The **crossfader** blends between Deck A and Deck B for seamless mood changes.
- A **Scene** is a saved photograph of your whole board — recall it to instantly
  restore what was playing and how.
